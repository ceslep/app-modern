# Prompt de Refactorización: Mejora UX/UI del Módulo de Notas

## 📋 Contexto

Tienes un componente JavaScript @notas.js que gestiona el registro de notas académicas. La funcionalidad está completa y funciona correctamente. Tu tarea es refactorizar **SOLO la interfaz gráfica y la experiencia de usuario**, manteniendo intacta toda la lógica de negocio.

---

## 🎯 Objetivo Principal

Transformar la interfaz actual en una experiencia moderna, elegante y profesional aplicando las mejores tendencias de diseño 2024-2025, con especial énfasis en:

- **Glassmorphism** (efectos de vidrio esmerilado)
- **Micro-interacciones** y animaciones fluidas
- **Feedback visual** inmediato y claro
- **Diseño responsive** y accesible
- **Jerarquía visual** clara

---

## 🎨 Requisitos de Diseño

### 1. Estética General

- **Glassmorphism**: Aplica efectos de backdrop-filter, blur, transparencias sutiles y bordes con gradientes
- **Paleta de colores**: Mantén el esquema actual (#543391 como primario) pero añade gradientes modernos y tonos complementarios
- **Tipografía**: Usa una escala tipográfica clara con pesos variados para jerarquía
- **Espaciado**: Sistema de espaciado consistente (usa múltiplos de 4px o 8px)
- **Sombras**: Sombras suaves y multicapa para profundidad

3. Animaciones y Transiciones

/* Definir sistema de animaciones */
:root {
  --transition-fast: 150ms cubic-bezier(0.4, 0, 0.2, 1);
  --transition-normal: 250ms cubic-bezier(0.4, 0, 0.2, 1);
  --transition-slow: 350ms cubic-bezier(0.4, 0, 0.2, 1);
}

/* Animaciones clave */
@keyframes fadeIn {
  from { opacity: 0; transform: translateY(10px); }
  to { opacity: 1; transform: translateY(0); }
}

@keyframes scaleIn {
  from { opacity: 0; transform: scale(0.95); }
  to { opacity: 1; transform: scale(1); }
}

@keyframes slideIn {
  from { opacity: 0; transform: translateX(-20px); }
  to { opacity: 1; transform: translateX(0); }
}
